# 版本迭代记录

## v2.2.0 (2026-04-15)

### 问题修复
- 🔧 **修复任务创建失败**：dueDate 改为可选字段，任务可独立创建无需关联计划
- 🔧 **修复任务审核工作流缺失**：添加 Draft → Pending → Active 完整流程
- 🔧 **修复权限矩阵错误**：员工无法创建/编辑/删除计划，经理无法删除计划和任务
- 🔧 **修复人员管理权限**：员工无权限操作用户管理

### 数据库变更
- Task 模型新增审核字段：`submittedAt`, `submittedById`, `reviewedAt`, `reviewedById`, `reviewComment`
- Task 状态枚举新增：`draft`, `pending`, `active`, `in_progress`, `completed`, `verified`

### API 变更
- `POST /api/tasks-v2` - 创建任务（草稿），dueDate 改为可选
- `POST /api/tasks-v2/:id/submit` - 提交审核
- `POST /api/tasks-v2/:id/withdraw` - 撤回审核
- `POST /api/tasks-v2/:id/review` - 审核任务（经理及以上）
- `DELETE /api/tasks-v2/:id` - 删除任务（经理及以上，仅草稿）

## v2.0.0 (2026-04-15)

### 新增功能
- ✨ **审核工作流**：Draft → Pending → Active 三阶段管控
- ✨ **变更申请机制**：已生效内容修改需提交申请
- ✨ **版本历史追踪**：ContentVersion 模型记录变更历史

### 权限控制增强
- 🔒 只有草稿状态可编辑/删除
- 🔒 员工无法删除任务
- 🔒 分层审核权限

### 数据库变更
- 新增 `ChangeRequest` 模型
- 新增 `ContentVersion` 模型
- Strategy/Plan 新增审核相关字段
